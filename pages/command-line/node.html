# 命令行更新node版本

  1. 清楚npm缓存，执行命令：
    npm cache clean -f
    1
  2. 安装n模块(专门用来管理nodejs版本的)：
    npm install -g n
    1
  3. 升级nodejs到最新的稳定版本
    n stable
    1
  4. 查看node版本以及node的安装路径
    node -v
<!-- v12.16.1 -->

npm -v
<!-- 6.13.4 -->

which node 
<!-- /usr/local/bin/nod -->
